# Changelog

All notable changes to this dataset will be documented in this file.

## [v2026.03] - 2026-03-04

### Added

- Initial public release
- 18 category breakdowns with revenue, pricing, and sales data
- 8 pricing tier analyses
- Platform comparison (Gumroad, Whop, Systeme.io, Payhip, Lemonsqueezy)
- Revenue concentration data (top 1%/5%/10%, bottom 50%)
- Seller distribution by portfolio size
- Review and rating impact analysis
- Product type breakdown (downloads, e-books, courses, bundles, memberships)
- Top 8 highest-grossing products
- Pay-What-You-Want vs fixed pricing comparison
- All data in both CSV and JSON formats
- Summary statistics in `analysis/summary-stats.json`

### Coverage

- 146,271 products
- 43,884 sellers
- 18 categories
- $206M+ estimated total revenue
