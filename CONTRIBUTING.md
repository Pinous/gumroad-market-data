# Contributing

Thanks for your interest in improving this dataset.

## Reporting Errors

If you spot an error in the data, please [open an issue](https://github.com/Pinous/gumroad-market-data/issues/new) with:

- Which file and row contains the error
- What the current value is
- What you believe the correct value should be
- Your source or reasoning

## Suggesting Additions

Want to see additional data points or breakdowns? Open an issue describing:

- What data you'd like to see
- How you'd use it
- Any sources you know of

## Data Updates

This dataset is updated monthly by [InsightRaider](https://insightraider.com). If you have access to complementary data sources and want to collaborate on a more comprehensive dataset, reach out via an issue.

## Attribution

If you use this data in a project, paper, or article, we'd love to hear about it. Drop a link in the [Discussions](https://github.com/Pinous/gumroad-market-data/discussions) tab.
